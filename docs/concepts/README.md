# 概念库

> 每个学习点的「概念 + 实践 + 踩坑」独立成篇。学一个记一个，不要攒。

## 使用说明

1. 学新知识点时，复制 [`_template.md`](_template.md) → 重命名 → 填内容
2. 实践踩坑写进 `⚠️ 踩坑记录`，同时在 [`答疑.md`](../学习答疑.md) 里加一条汇总
3. 完成后回到 [`学习路线.md`](../学习路线.md) 勾 checkbox
