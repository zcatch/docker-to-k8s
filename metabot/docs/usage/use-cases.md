# Use Cases

## Solo AI Developer

Full Claude Code from your phone, bound to your project. Read, edit, run tests, commit — all from Feishu or Telegram.

## Multi-Agent Team

Frontend bot, backend bot, infra bot, each in their own workspace, talking via the Agent Bus. Delegate tasks across bots with `mb talk`.

## Self-Growing Organization

A manager bot that creates new agents on demand via MetaSkill, assigns tasks, schedules follow-ups. The organization grows as needed.

## Autonomous Research Pipeline

Agents that search, analyze, save findings to MetaMemory, and schedule next steps. Build knowledge bases that grow automatically.

## Voice Assistant (Jarvis Mode)

"Hey Siri, Jarvis" from AirPods — hands-free voice control of any agent via iOS Shortcuts. Server-side Doubao/Whisper STT for high-quality speech recognition. See the [Voice Setup Guide](../features/voice-jarvis.md).

## Web Phone Call Mode

Open the [Web UI](../features/web-ui.md) and tap the phone icon for hands-free voice conversation directly in your browser. VAD auto-detects when you finish speaking, auto-cycles record → reply → record like a real phone call. Works on desktop and mobile (HTTPS required for mobile microphone access).
