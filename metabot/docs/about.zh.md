# 关于

MetaBot 由 [XVI Robotics](https://xvirobotics.com) 打造，我们做人形机器人大脑。

我们在内部用 MetaBot 把公司打造成 **Agent Native 组织** — 一个小团队的人类，监督自我进化的 AI Agent。

我们开源它，因为我们相信这是未来公司的运行方式。
